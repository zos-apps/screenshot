## About

Capture screenshots and screen recordings.

## Features

- Full screen capture
- Window capture
- Selection capture
- Recording
- Annotations

## Installation

### From App Store
Open the **App Store** in zOS and search for "Screenshot".

### Via npm
```bash
npm install @zos-apps/screenshot
```

## Usage

- Choose capture mode
- Click or drag to capture
- Edit and save

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+3` | Full screen |
| `Cmd+Shift+4` | Selection |
| `Cmd+Shift+5` | Options |


## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
